*** How to migrate Markdown files to Emacs org mode
#+begin_src bash
for f in *.md; do pandoc --wrap=preserve -f markdown -t org ${f} | sed -E "/^[[:blank:]]*:/d" > ${f%%.md}.org; done
#+end_src
