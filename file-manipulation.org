* File Manipulation
** Images
*** Add metadata to PNG
#+begin_example
convert IN.png \
        -set 'Copyright' 'CC-BY-SA 4.0' \
        -set 'Title' 'A wonderful day' \
        -set comment 'Photo taken while running' \
        OUT.png
#+end_example

*** Mirror images
Convert uses =-flop= for horizontal mirroring, and =-flip= for vertical

#+begin_example
convert -flop input.png output.png
#+end_example
