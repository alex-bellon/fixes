# File Conversion

## CHM to PDF

```
chm2pdf OPTIONS FILENAME.CHM FILENAME.PDF
```
[Source](https://askubuntu.com/questions/706012/how-to-convert-this-chm-file-to-pdf)

## GIF to Image frame

```
convert file.gif frame.png
```

## Image to PDF

```
convert image.png out.pdf
```

## JPG to PNG
```
mogrify -format png pic.jpg
```

## Text to hex
```
echo 'hello' | xxd -
```
